# Ali Mushkil Kusha — alimushkilkusha.com

> **The Solver of Difficulties** — A comprehensive website dedicated to the life, wisdom, and eternal legacy of Hazrat Ali ibn Abi Talib (RA).

---

## 🌙 About

This website serves as the definitive English/Urdu resource on Hazrat Ali (RA), covering his biography, sayings (Aqwal-e-Zareen), Nahj al-Balagha, family lineage, military career, governance principles, and much more.

## 🎨 Design Theme: "Noor-e-Ali" (Light of Ali)

### Color Palette
| Role | Color | Hex |
|------|-------|-----|
| Primary (Emerald) | Deep green | `#064635` |
| Secondary (Gold) | Rich gold | `#C8A951` |
| Accent (Navy) | Royal blue | `#1B2A4A` |
| Background | Warm ivory | `#FBF7F0` |
| Dark Sections | Charcoal | `#1A1A2E` |

### Typography
- **Headings:** Playfair Display (elegant serif)
- **Body:** Inter (clean sans-serif)
- **Arabic/Urdu:** Amiri, Noto Nastaliq Urdu
- **Decorative:** Aref Ruqaa

### Design Elements
- Islamic geometric patterns as backgrounds & dividers
- Arabesque ornamental borders
- Gold radial gradients evoking divine light
- Animated golden particle effects
- Crescent & star brand element

## 🛠️ Technology

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — No frameworks for maximum compatibility
- **Google Fonts** — Web Typography
- **Responsive** — Mobile-first, works on all devices
- **Accessible** — Reduced motion support, semantic HTML, ARIA labels

## 🚀 Deployment

This is a static website — simply upload all files to your GoDaddy hosting via FTP or file manager. No build step required.

1. Upload all files to the `public_html` directory on your hosting
2. Ensure `index.html` is in the root
3. The website is live!
